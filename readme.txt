Adds a customizable Elementor widget to embed Wistia videos with overlay play button, color, size, border and radius styling.

== Description ==
This plugin provides a simple Elementor widget for embedding Wistia videos with a custom play button overlay.  
You can change the play button's icon, size, color, border, and radius directly from Elementor.

== Installation ==
1. Upload the plugin to `/wp-content/plugins/`.
2. Activate the plugin.
3. In Elementor editor, search for **Wistia Video** widget.

== Frequently Asked Questions ==
= Do I need Elementor? =
Yes, Elementor must be installed and active.
